About

About this app

This workspace is a lightweight browser app built as a self-contained control surface. The goal is to make the home, about, and dashboard experiences work together without any external build tooling.

What it does

The app combines a landing page, an informational page, and an interactive dashboard.

  • Home page Introduces the app and links directly into the dashboard.
  • About page Explains the purpose of the workspace and how the views are organized.
  • Dashboard Lets you change controls, filter assets, and review the audit trail.

How it runs

A single HTML file handles the routing and behavior, so it stays simple to run on Windows.

  • Navigation Hash-based routing switches between views without reloading the app.
  • State Control settings and filters persist in localStorage.
  • Server Use serve.ps1 to run the app locally if you want a local server.
